# Контент-завод для бренда Мария — разбор по шагам

Это документ о том, как устроена наша система изнутри.
Простыми словами — что делает каждый шаг и какими сервисами это достигается.

---

## Шаг 1 — Поиск вирусных видео

**Задача:** найти видео, которые прямо сейчас набирают много просмотров по теме натуральной косметики, ухода за кожей, красоты.

Нас интересуют не просто популярные видео — а те, что **быстро растут**. Это значит тема горячая прямо сейчас.

### Где ищем и чем:

**YouTube**
Официальное API от Google. Позволяет искать видео по ключевым словам, фильтровать по дате загрузки, сортировать по количеству просмотров. Бесплатно до определённого объёма запросов.
→ *Используем: YouTube Data API v3*

**TikTok**
TikTok не даёт официального доступа для таких задач. Поэтому используем сторонний сервис, который умеет собирать данные из TikTok.
→ *Используем: Apify (платный парсер, ~$5/мес)*

**Reddit**
Огромное сообщество, где люди обсуждают косметику, делятся советами, спрашивают рекомендации. Там можно найти темы, которые реально волнуют людей. Reddit даёт бесплатный публичный доступ к своим данным.
→ *Используем: Reddit API (бесплатно)*

**Google Trends**
Показывает, что люди ищут в Google прямо сейчас. Можно понять, какие темы на подъёме.
→ *Используем: Google Trends (через pytrends, бесплатно)*

---

## Шаг 2 — Скачивание видео

**Задача:** скачать найденное видео на наш сервер, чтобы дальше его обработать.

Это самая универсальная часть — один инструмент умеет скачивать видео с тысяч сайтов: YouTube, TikTok, Instagram, Vimeo, Reddit и многих других. Не нужно отдельное решение для каждой платформы.

→ *Используем: yt-dlp — бесплатная программа с открытым кодом*

---

## Шаг 3 — Транскрибация (перевод речи в текст)

**Задача:** превратить то, что говорится в видео, в текст. Без этого ИИ не сможет понять содержание.

Видео может быть на любом языке — русском, английском, корейском. Нам это не помеха.

### Варианты:

**Whisper от OpenAI**
Самый точный инструмент на сегодня. Понимает 99 языков. Работает даже с акцентом и фоновым шумом. Платим только за объём — примерно $0.006 за минуту аудио (очень дёшево).
→ *Основной вариант*

**AssemblyAI**
Альтернатива с дополнительными возможностями: автоматически разделяет текст по спикерам, умеет определять темп речи и эмоции. Чуть дороже Whisper.
→ *Используем как опцию для глубокого анализа*

---

## Шаг 4 — Анализ вирусного ДНК

**Задача:** понять, почему конкретное видео набрало миллионы просмотров. Что в нём такого?

Это самый важный и умный шаг. ИИ читает расшифрованный текст и находит:

- **Крючок** — что происходит в первые 3 секунды, почему не хочется листать дальше
- **Боль или желание** — какую проблему или мечту это видео затрагивает
- **Структура** — как построен рассказ, чтобы люди досматривали до конца
- **Эмоция** — что человек чувствует: удивление, узнавание себя, страх упустить
- **Призыв** — к чему в итоге подталкивает видео

### Чем анализируем:

**Claude от Anthropic**
Один из лучших ИИ для глубокого анализа текстов и понимания смысла. Отлично работает на русском языке. Умеет давать структурированные ответы по заданным критериям.
→ *Основной инструмент для анализа*

**GPT-4 от OpenAI**
Альтернатива Claude. Тоже очень сильный, можно использовать как резервный вариант.
→ *Запасной вариант*

## Шаг 5 — Написание сценария

**Задача:** на основе анализа написать готовый сценарий для нашего видео про косметику Мария.

ИИ использует всё что узнал на предыдущем шаге и создаёт текст, который:
- Использует ту же структуру что и вирусное видео
- Говорит про наш продукт
- Написан голосом и стилем бренда Мария
- Адаптирован под нужный формат и длину

Для каждой темы сразу пишется несколько вариантов:
- Короткий (до 30 секунд) — для TikTok
- Средний (до 60 секунд) — для Instagram Reels
- Длинный (1–2 минуты) — для YouTube Shorts
- Полноценный (5–15 минут) — для YouTube

→ *Используем: Claude от Anthropic*

## Шаг 6 — Создание видео

**Задача:** превратить текст сценария в готовый видеоролик.

Это состоит из нескольких частей:

### Голос
Текст озвучивается синтетическим голосом. Современные технологии делают голос живым — с интонациями, паузами, эмоциями. Можно создать фирменный голос бренда.

**ElevenLabs**
Лучший сервис для русского синтеза речи. Звучит как живой человек. Можно клонировать реальный голос — если записать 10–15 минут своего голоса, сервис будет говорить им.
→ *Основной вариант*

**Yandex SpeechKit**
Российская альтернатива. Хорошо работает с русским языком, есть много голосов.
→ *Запасной вариант*

### Аватар (виртуальный ведущий)
ИИ-персонаж, который читает сценарий на камеру. Выглядит как реальный человек.

**HeyGen**
Сервис для создания ИИ-аватаров. Можно выбрать готового персонажа или загрузить своё видео и создать цифрового двойника. Аватар двигает губами в такт голосу, моргает, жестикулирует.
→ *Используем для видео с ведущим*

### Дополнительные клипы (B-roll)
Красивые вставки между речью ведущего: текстуры крема, цветы, природа, процесс нанесения. Делают видео живым и визуально богатым.

**Runway ML**
ИИ-генератор видеоклипов. Описываешь что хочешь увидеть — сервис создаёт короткий клип.
→ *Используем для B-roll вставок*

**Pexels / Pixabay**
Бесплатные библиотеки стоковых видео. Тысячи клипов по теме красоты и косметики — бесплатно и сразу.
→ *Используем как дополнительный источник клипов*

### Финальная сборка
Все части — голос, аватар, B-roll, музыка, субтитры — склеиваются в один файл.

**FFmpeg**
Профессиональный бесплатный инструмент для работы с видео. Именно он делает финальный монтаж по нашим инструкциям.
→ *Используем для сборки*

## Шаг 7 — Написание статей

**Задача:** создавать текстовые материалы для блога, Дзена, Telegram.

ИИ пишет полноценную статью с заголовком, вступлением, разделами и выводом. Мы задаём тему, тон и длину.

Примеры статей для Марии:
- «Почему натуральная косметика лучше — разбираем состав»
- «5 ошибок в уходе за кожей, которые вы делаете каждый день»
- «Как выбрать крем для лица: полный гайд»

→ *Используем: Claude от Anthropic*

---

## Шаг 8 — Написание постов

**Задача:** короткие тексты для ежедневного постинга в Instagram, VK, Telegram.

ИИ пишет несколько вариантов одного поста в разных стилях: информационный, личный, с юмором, с вопросом к аудитории. Выбираем лучший.

→ *Используем: Claude от Anthropic*

---

## Шаг 9 — Карусели для Instagram

**Задача:** серия слайдов с текстом и картинкой, которые листают пальцем.

Карусели отлично работают для обучающего контента — люди сохраняют их и возвращаются.

Примеры для Марии:
- «7 токсичных ингредиентов в косметике — слайд за слайдом»
- «Утренний уход: шаг 1, шаг 2, шаг 3...»
- «Как читать состав на упаковке»

Как это работает:
- ИИ пишет текст для каждого слайда (заголовок + 2–3 строки)
- Программа собирает слайды в шаблон с цветами и шрифтами бренда Мария
- Получаем готовый набор картинок для публикации

→ *Используем: Claude (текст) + шаблоны в стиле бренда*

# Content Factory — Roadmap разработки

## Текущий статус

### Работает полностью
- Пайплайн: скачивание → транскрипция → анализ → генерация сценариев → сборка видео
- Дискавери: YouTube (YouTube Data API v3), Reddit (публичный API)
- Dashboard: 3 вкладки (Discover / Process / Results), polling задач
- Голос (ElevenLabs), аватар (HeyGen), сборка (FFmpeg)

### Работает частично
- TikTok Discovery — код готов, нужен Apify API ключ ($5/мес)
- Runway B-roll — модуль есть, не подключён к пайплайну
- База данных — модели определены, данные не сохраняются

### Не реализовано
- Persistence результатов в БД
- Retry-логика при ошибках в Celery
- История обработанных видео
- Docker / деплой
- Автопубликация

---

## Фаза 1 — Стабилизация (текущий приоритет)

Цель: сделать систему надёжной и не терять данные.

- [ ] Сохранение результатов пайплайна в SQLite/Postgres через SQLAlchemy
- [ ] Retry-логика для каждого этапа Celery (скачивание, транскрипция, генерация)
- [ ] Подключить Runway B-roll в финальную сборку FFmpeg
- [ ] Протестировать TikTok Discovery с Apify ключом
- [ ] Страница истории: все обработанные видео с результатами

---

## Фаза 2 — Продуктовые фичи

Цель: улучшить UX и расширить возможности.

- [ ] Редактор сценариев — правка перед озвучкой прямо в UI
- [ ] Пакетная обработка — загрузить несколько URL сразу
- [ ] Мультибрендовость — не только Марья, настраиваемый бренд
- [ ] Предпросмотр сценария с таймингами
- [ ] Экспорт в Google Docs / Notion

---

## Фаза 3 — Инфраструктура и деплой

Цель: запустить в продакшене.

- [ ] Docker Compose: FastAPI + Celery + Redis + Postgres
- [ ] Переменные окружения: полная документация всех ключей
- [ ] Health monitoring: алерты при падении воркера
- [ ] Nginx / reverse proxy конфиг
- [ ] CI/CD (GitHub Actions)

---

## Фаза 4 — Расширение

Цель: полная автоматизация публикации и аналитики.

- [ ] Автопубликация в TikTok / Instagram Reels / YouTube Shorts
- [ ] Дашборд аналитики: просмотры, вовлечённость по опубликованным видео
- [ ] A/B тестирование сценариев
- [ ] Webhook-уведомления (Telegram-бот о готовом видео)

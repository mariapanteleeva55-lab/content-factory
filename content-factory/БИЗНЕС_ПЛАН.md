# Content Factory — план работы

## Для чего мы это строим

Это личный инструмент для бренда натуральной косметики **Мария**.
Цель — производить большой объём качественного контента быстро, без найма большой команды.

Мы будем создавать пять типов контента:
1. Короткие вирусные видео (TikTok, Instagram Reels, YouTube Shorts)
2. Длинные видео (YouTube — обзоры, гайды, сторителлинг)
3. Статьи (для блога, Дзена, Telegram)
4. Обычные посты (Instagram, VK, Telegram)
5. Посты-карусели (Instagram — несколько слайдов с текстом и картинкой)

---

## Как будет работать каждый тип контента

---

### 1. Короткие вирусные видео

**Шаг 1 — Находим что сейчас набирает просмотры**
Программа автоматически заходит на YouTube, TikTok и Reddit и собирает видео по теме косметики, натуральных средств, ухода за кожей. Отбираем не просто популярные, а те, что быстро растут прямо сейчас — это значит тема «горячая» и людям это интересно сегодня.

*Инструменты: YouTube Data API (официальный поиск YouTube), Apify (просмотр TikTok), Reddit API (форумы и обсуждения)*

**Шаг 2 — Скачиваем и изучаем видео**
Программа скачивает выбранное видео и переводит речь в текст — на любом языке.

*Инструменты: yt-dlp (скачивание видео с тысяч сайтов), Whisper от OpenAI (перевод речи в текст, понимает 99 языков)*

**Шаг 3 — Понимаем почему видео зашло**
Искусственный интеллект читает текст и описание видео и отвечает на вопросы: с чего начинается видео? какую боль или желание оно задевает? почему хочется досмотреть до конца? какова структура?

*Инструмент: Claude от Anthropic (один из лучших ИИ для анализа и написания текстов)*

**Шаг 4 — Пишем сценарий для Марии**
На основе анализа ИИ пишет готовый сценарий — уже про нашу косметику, нашим голосом, для нашей аудитории. Сразу три варианта под разные платформы: короткий для TikTok, средний для Reels, чуть длиннее для Shorts.

**Шаг 5 — Записываем видео**
- Голос озвучивает текст — звучит как живой человек, не робот
- ИИ-аватар (виртуальный ведущий) читает сценарий на камеру
- Программа автоматически склеивает всё в готовый ролик с музыкой

*Инструменты: ElevenLabs (синтез голоса на русском), HeyGen (ИИ-аватар), FFmpeg (финальная сборка видео)*

---

### 2. Длинные видео для YouTube

Это не просто «вирусный клип», а полноценное видео на 5–15 минут.
Форматы: обзор продукта, гайд по уходу за кожей, история бренда, «до и после», FAQ.

**Как будет работать:**
- Программа ищет популярные длинные видео конкурентов или смежных тем
- Анализирует структуру: вступление, основные блоки, призыв к действию
- ИИ пишет подробный сценарий с таймингами и указаниями для монтажа
- Голос озвучивает, аватар ведёт, программа монтирует

*Дополнительно:* система сможет добавлять B-roll клипы — красивые вставки с природой, текстурами, процессом нанесения крема и т.д.

*Инструмент для B-roll: Runway (генерация коротких видеоклипов через ИИ)*

---

### 3. Статьи

Статьи нужны для блога на сайте, Яндекс Дзена, Telegram-канала, SEO.

**Как будет работать:**
- Задаём тему (например: «5 причин перейти на натуральный крем»)
- ИИ изучает что уже написано по этой теме — что популярно, что ищут люди
- Пишет готовую статью: с заголовком, подзаголовками, вступлением, выводами
- Текст можно сразу редактировать прямо в нашей программе перед публикацией

*Инструмент: Claude (написание длинных структурированных текстов — его сильная сторона)*

---

### 4. Обычные посты

Короткие тексты для Instagram, VK, Telegram.
Форматы: анонс продукта, совет по уходу, цитата, факт о составе.

**Как будет работать:**
- Пишем тему или кидаем ссылку на продукт
- ИИ пишет несколько вариантов поста с разным тоном: серьёзный, дружеский, с юмором
- Выбираем понравившийся, при желании правим
- Программа подбирает подходящие хэштеги

*Инструмент: Claude*

---

### 5. Посты-карусели для Instagram

Карусель — это несколько слайдов, которые листают пальцем. Очень хорошо работают для обучающего контента.
Примеры: «7 ингредиентов которых стоит избегать», «Утренний уход — шаг за шагом», «Как читать состав крема».

**Как будет работать:**
- Задаём тему
- ИИ пишет текст для каждого слайда: заголовок + 2–3 строки
- Программа собирает слайды в готовый шаблон с нашими цветами и шрифтами бренда
- Выгружаем набор картинок — можно сразу публиковать

*Инструменты: Claude (текст) + генерация изображений или готовые шаблоны (Canva API / собственные шаблоны в фирменном стиле)*

## Дорожная карта — что и когда делаем

---

### Этап 1 — Короткие видео работают стабильно
*Это уже почти готово, нужно дошлифовать*

Что доделываем:
- [ ] Все результаты сохраняются — не теряются если закрыть браузер
- [ ] История: видим все что уже сделали раньше
- [ ] Если что-то пошло не так (интернет отвалился, сервис не ответил) — программа сама повторяет попытку
- [ ] TikTok: подключаем поиск по TikTok полностью
- [ ] B-roll: вставляем красивые клипы в финальное видео

**Итог этапа:** нажал кнопку → получил готовый короткий ролик → скачал → опубликовал

---

### Этап 2 — Добавляем длинные видео для YouTube

Что делаем:
- [ ] Поиск популярных длинных видео по теме косметики
- [ ] Анализ структуры длинного видео
- [ ] Написание подробного сценария с таймингами
- [ ] Голос + аватар + B-roll + монтаж длинного формата
- [ ] Автоматические субтитры

**Итог этапа:** можем делать полноценные YouTube-ролики на 5–15 минут

---

### Этап 3 — Добавляем тексты (статьи и посты)

Что делаем:
- [ ] Генерация статей: задаём тему → получаем готовый текст
- [ ] Генерация постов: несколько вариантов тона, выбор лучшего
- [ ] Редактор текста прямо в программе — правим перед публикацией
- [ ] Хэштеги подбираются автоматически
- [ ] История всех написанных текстов — можно вернуться и переработать

**Итог этапа:** один инструмент закрывает весь текстовый контент бренда

---

### Этап 4 — Добавляем карусели для Instagram

Что делаем:
- [ ] Задаём тему → ИИ пишет текст для 7–10 слайдов
- [ ] Программа собирает слайды в шаблон с брендовыми цветами и шрифтами Марии
- [ ] Готовые картинки скачиваем одним архивом
- [ ] Библиотека шаблонов: разные стили (обучающий, вдохновляющий, продуктовый)

**Итог этапа:** карусели с единым визуальным стилем бренда — без дизайнера

---

### Этап 5 — Автопубликация и планирование

Что делаем:
- [ ] Контент-план: видим что запланировано на неделю/месяц
- [ ] Ставим дату и время → готовый пост публикуется сам
- [ ] Поддерживаемые платформы: Instagram, TikTok, YouTube, Telegram, VK
- [ ] Уведомление в Telegram когда контент опубликован

**Итог этапа:** создали контент → запланировали → всё публикуется само по расписанию

---

## Итог: что получим в конце

Один инструмент, который делает всё:

| Тип контента | Платформа | Степень автоматизации |
|---|---|:---:|
| Короткое вирусное видео | TikTok, Reels, Shorts | Полностью авто |
| Длинное видео | YouTube | Авто + лёгкая правка |
| Статья | Блог, Дзен, Telegram | Авто + лёгкая правка |
| Пост | Instagram, VK, Telegram | Авто + выбор варианта |
| Карусель | Instagram | Авто + готовые слайды |

Вместо отдельных сценаристов, редакторов, дизайнеров и монтажёров —
один инструмент, который работает быстро и в едином стиле бренда Мария.
